# Crazy Monkey — 90-second pitch script

Use this before every restaurant owner demo. Two devices: **customer phone** + **kitchen tablet**.

## Setup (2 minutes before)

1. Open **`/demo`** on your laptop — QR + PIN cheat sheet.
2. Kitchen tablet → `/staff` → PIN **1111** → Live Board.
3. Optional third screen: Owner laptop → PIN **3333**.
4. If leftover orders from last pitch: Owner → **Reset Demo**.
5. Confirm WiFi / phone hotspot works on both devices.

## The 90-second reveal

| Time | Say | Do |
|------|-----|----|
| 0:00 | "Customer scans the table QR — no app download." | Show phone scanning `/demo` QR → Welcome |
| 0:15 | "They pick Chinese, Italian, or dessert — add a couple of dishes." | Add 2 items → cart → **Place Order — Pay at Counter** |
| 0:35 | "Watch the kitchen." | Point at tablet — new ticket appears in **New Orders** (chime) |
| 0:45 | "Kitchen starts it…" | Tap **START** → moves to Preparing |
| 0:55 | "…and marks ready." | Tap **READY** → customer tracking steps update live |
| 1:05 | "Owner sees the numbers update in real time." | Owner dashboard KPIs refresh |
| 1:20 | "Stock manager tracks ingredients separately — no guesswork." | Optional: PIN **2222** glance at low-stock |
| 1:30 | "That's the whole loop. Reset and we can run it again." | |

## After the pitch

- Owner → **Reset Demo** so the next meeting starts clean.
- Kitchen board should show exactly **1 New** + **1 Preparing** after reset.
